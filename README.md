# 🐶 Max Studio OS

AI-powered dog training video production system for Clear Mind Canine.

---

## 🎬 System Purpose
Turn structured training scripts into automated AI video episodes.

---

## ⚙️ Pipeline
Notion → Python → ElevenLabs → Runway/Pika/Luma → FFmpeg → YouTube

---

## 🧠 Core Doctrine
- No verbal cues during shaping
- Stay calm, patient, and pleasant
- Click quickly
- Reward generously
- Caroselling = engagement, not problem
- Use space instead of force

---

## 📁 Structure
- src/ → automation code
- episodes/ → video projects
- config/ → training rules
